\section{Durchführung}
\label{sec:Durchführung}

Nicht jedes Protokoll muss in der Reihenfolge
\begin{itemize}
  \item Theorie
  \item Durchführung
  \item Auswertung
  \item Diskussion
\end{itemize}
geschrieben werden.
Wenn es der Versuch zu lässt, ist zum Beispiel auch Folgendes möglich:
\begin{itemize}
  \item Theorie
  \item Durchführung
  \item Material 1
  \begin{itemize}
    \item Auswertung
    \item Diskussion
  \end{itemize}
  \item Material 2
  \begin{itemize}
    \item Auswertung
    \item Diskussion
  \end{itemize}
\end{itemize}
